# Two-Minute Demo

Goal: show Practiq as a calm daily practice workflow, not just a notes app.

## Flow

1. Open the dashboard.
   - "Practiq is a calmer practice system for independent healthcare practitioners."

2. Show today&rsquo;s schedule.
   - "You can start from appointments, but Practiq also supports visits without appointments."

3. Open a patient record.
   - "Patient demographics, visit history, forms, communication, and notes stay connected."

4. Start or view a Visit Note.
   - "Simple Visit Note Mode gives practitioners a natural writing surface."
   - "SOAP / Insurance Mode is available when structured documentation is needed."

5. Mention AI assist.
   - "AI can help improve notes, draft emails, shape reminders, and support import, but the practitioner reviews everything."

6. Show intake/checkout/import/export briefly.
   - "Practiq supports the surrounding workflow too: intake, checkout, billing support, import, and export."
